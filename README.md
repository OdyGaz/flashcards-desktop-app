# Local Flashcards Desktop Application

Μια σύγχρονη, τοπική εφαρμογή desktop για την εκμάθηση γλωσσών (ή άλλων αντικειμένων) με τη χρήση **Flashcards (καρτών)**. Η εφαρμογή επιτρέπει στον χρήστη να οργανώνει κάρτες με εικόνες σε Decks, να πληκτρολογεί την απάντησή του και να λαμβάνει άμεσο οπτικό feedback για τυχόν ορθογραφικά λάθη.

---

## 🚀 Βασικά Χαρακτηριστικά

* **Διαχείριση Decks**: Δημιουργία, προβολή και διαγραφή decks καρτών τοπικά.
* **Δημιουργία Καρτών με Δύο Τρόπους**:
  * **Αναζήτηση Αρχείου (Browse)**: Επιλογή οποιασδήποτε εικόνας από τον υπολογιστή σας.
  * **Επικόλληση από το Πρόχειρο (Clipboard Paste)**: Άμεση εισαγωγή εικόνας με screenshot (π.χ. `Win + Shift + S` στα Windows) και πάτημα του κουμπιού επικόλλησης.
* **Έξυπνος Έλεγχος & Feedback Σφαλμάτων**: 
  * Χρήση του αλγορίθμου `difflib` για σύγκριση χαρακτήρα προς χαρακτήρα.
  * Οπτική επισήμανση (με έντονο κοραλί χρώμα) των γραμμάτων που λείπουν ή είναι λάθος στην προσπάθεια του χρήστη.
* **Σύστημα Βαθμολογίας (Score Tracking)**:
  * **+1 πόντος** για κάθε σωστή απάντηση.
  * **-1 πόντος** για κάθε λανθασμένη απάντηση.
* **Γρήγορη Ροή με το Enter**: 
  * 1ο πάτημα του **Enter**: Έλεγχος απάντησης και εμφάνιση feedback.
  * 2ο πάτημα του **Enter**: Αυτόματη μετάβαση στην επόμενη κάρτα.
* **Σύγχρονη Dark Mode Αισθητική**: Neumorphic σχεδίαση βασισμένη σε custom Qt Style Sheets (QSS).

---

## 🛠️ Τεχνολογικό Stack

* **Γλώσσα Προγραμματισμού**: Python (Συμβατότητα με εκδόσεις >= 3.10 έως 3.14)
* **Γραφικό Περιβάλλον (UI)**: [PySide6](https://doc.qt.io/qtforpython-6/) (Επίσημα Python bindings για το Qt Framework)
* **Διαχείριση Βάσης Δεδομένων**: [SQLAlchemy 2.0](https://www.sqlalchemy.org/) ORM με τοπική βάση **SQLite**
* **Επεξεργασία Εικόνας**: [Pillow (PIL)](https://python-pillow.org/)
* **Αλγόριθμος Σύγκρισης**: `difflib` (Ενσωματωμένη βιβλιοθήκη Python)

## 📁 Δομή Φακέλων (Architecture)

Η εφαρμογή ακολουθεί τις αρχές της **Layered Architecture (Αρχιτεκτονική Επιπέδων)** για καθαρό διαχωρισμό ευθυνών (Separation of Concerns):

```text
flashcards_app/
│
├── data/                           # Τοπικός φάκελος δεδομένων (SQLite & εικόνες)
│
├── src/                            # Ο πηγαίος κώδικας της εφαρμογής
│   ├── domain/                     # Domain Layer (Μοντέλα δεδομένων SQLAlchemy)
│   ├── repositories/               # Data Access Layer (CRUD λειτουργίες βάσης)
│   ├── services/                   # Business Logic (Λογική μελέτης, σκορ, επεξεργασίας εικόνας)
│   └── ui/                         # Presentation Layer (PySide6 παράθυρα, views & styles)
│
└── pyproject.toml                  # Ρυθμίσεις εξαρτήσεων Poetry
```

## ⚙️ Εγκατάσταση και Εκτέλεση

### Απαίτηση Συστημάτων
* Python εγκατεστημένη στο σύστημα (εκδόσεις `3.10` έως `3.14.x`).

---

### Επιλογή Α: Εγκατάσταση με Virtual Environment (venv - Συνιστάται για γρήγορη εκκίνηση)

Ανοίξτε το τερματικό σας στον κεντρικό φάκελο του project και εκτελέστε:

```bash
# 1. Δημιουργία εικονικού περιβάλλοντος
python -m venv .venv

# 2. Ενεργοποίηση του περιβάλλοντος
# Στα Windows (CMD/PowerShell):
.venv\Scripts\activate
# Σε Mac/Linux:
source .venv/bin/activate

# 3. Αναβάθμιση του pip και εγκατάσταση εξαρτήσεων
python -m pip install --upgrade pip
pip install PySide6 SQLAlchemy Pillow

# 4. Εκτέλεση της εφαρμογής
python -m src.main
```

---

### Επιλογή Β: Εγκατάσταση με Poetry (Διαχείριση Πακέτων)

Αν προτιμάτε τη χρήση του Poetry, βεβαιωθείτε ότι το `pyproject.toml` περιλαμβάνει τις σωστές εκδόσεις και εκτελέστε:

```bash
# 1. Εγκατάσταση των εξαρτήσεων στο εικονικό περιβάλλον του Poetry
poetry install

# 2. Εκτέλεση της εφαρμογής
poetry run python -m src.main
```

## 📝 Πώς να χρησιμοποιήσετε την εφαρμογή

1. **Δημιουργήστε ένα Deck**: Πληκτρολογήστε ένα όνομα στο κάτω μέρος της αρχικής οθόνης και πατήστε "Δημιουργία Deck".
2. **Προσθέστε Κάρτες**: 
   * Επιλέξτε το Deck σας και πατήστε "Προσθήκη Καρτών".
   * Κάντε browse μια εικόνα ή τραβήξτε ένα screenshot (`Win+Shift+S`) και πατήστε "Επικόλληση".
   * Πληκτρολογήστε τη λέξη-στόχο στο πεδίο κειμένου και πατήστε "Αποθήκευση Κάρτας".
3. **Έναρξη Μελέτης**: Επιλέξτε το Deck στην αρχική οθόνη και πατήστε "Έναρξη Μελέτης".
4. **Μελέτη & Feedback**: 
   * Δείτε την εικόνα, γράψτε την απάντησή σας στο πλαίσιο και πατήστε **Enter** για έλεγχο.
   * Δείτε το σκορ σας να αλλάζει σε πραγματικό χρόνο και παρατηρήστε τα κόκκινα γράμματα που σας δείχνουν πού ακριβώς κάνατε λάθος.
   * Πατήστε ξανά **Enter** για να προχωρήσετε στην επόμενη κάρτα.
